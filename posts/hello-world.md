# Hello World

欢迎来到我的博客！这是使用纯静态方案搭建的第一篇文章。

## 为什么要搭建这个博客

作为一个开发者，拥有自己的博客是一件很重要的事情。在这里我可以：

- **自由地**记录和分享技术心得
- 不受第三方平台的限制
- 完全掌控内容和样式

## 技术方案

这个博客系统的特点：

1. **纯静态** - 没有后端，没有数据库
2. **Markdown 写作** - 用最简单的方式写文章
3. **GitHub Pages 托管** - 免费且稳定
4. **GitHub Actions 自动化** - 提交 `.md` 文件即可自动更新

### 代码示例

```javascript
// 获取博客文章列表
fetch('/assets/json/posts.json')
    .then(res => res.json())
    .then(posts => {
        console.log('Total posts:', posts.length);
    });
```

> 写作是最好的思考方式。通过写作，我们可以把模糊的想法变成清晰的逻辑。
