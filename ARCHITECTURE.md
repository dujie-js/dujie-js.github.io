# 📐 博客系统架构说明

## 🏗️ 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                      用户访问入口                         │
├─────────────────────────────────────────────────────────┤
│  index.html (主页) → 点击"博客"按钮 → blog.html         │
└─────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────┐
│                   博客列表页 (blog.html)                 │
├─────────────────────────────────────────────────────────┤
│  1. 加载 blog.js                                         │
│  2. 读取 posts.json                                      │
│  3. 渲染文章列表（分页）                                 │
│  4. 显示翻页按钮                                         │
└─────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────┐
│                  文章详情页 (post.html)                  │
├─────────────────────────────────────────────────────────┤
│  1. 从 URL 获取文章 ID                                   │
│  2. 加载 post.js                                         │
│  3. 从 posts.json 查找文章                               │
│  4. Markdown → HTML (marked.js)                         │
│  5. 渲染文章内容                                         │
└─────────────────────────────────────────────────────────┘
```

## 🎯 核心功能模块

### 1. 文章加载模块

```
blog.js: loadBlogPosts()
├── fetch('posts.json')
├── 解析 JSON
├── 按日期排序
├── 计算总页数
└── 调用 renderBlogList()
```

### 2. 分页模块

```
blog.js: renderPagination()
├── 计算当前页范围
├── 生成页码按钮
│   ├── 上一页
│   ├── 页码 [1] [2] [...] [n]
│   └── 下一页
└── 绑定点击事件
```

### 3. 渲染模块

```
blog.js: renderBlogList()
├── 计算当前页文章
│   └── slice(startIndex, endIndex)
├── 生成 HTML
│   ├── 标题
│   ├── 元信息（日期、作者、阅读时间）
│   ├── 标签
│   ├── 摘要
│   └── "阅读全文"链接
└── 插入 DOM
```

### 4. Markdown 渲染模块

```
post.js: renderPost()
├── 查找文章数据
├── marked.parse(content)
│   ├── # → <h1>
│   ├── ## → <h2>
│   ├── **text** → <strong>
│   ├── `code` → <code>
│   └── ```block``` → <pre><code>
└── 应用样式
```

## 🌐 URL 路由架构

```
网站根目录
│
├─ /                          → index.html (主页)
│   └─ 导航按钮"博客" →
│
├─ /blog.html                 → 博客列表（第1页）
│   ├─ ?page=1                → 第1页
│   ├─ ?page=2                → 第2页
│   └─ ?page=n                → 第n页
│       └─ 点击文章 →
│
└─ /post.html?id=xxx          → 文章详情
    ├─ ?id=post-001           → 文章1
    ├─ ?id=post-002           → 文章2
    └─ ?id=post-xxx           → 文章n
        └─ "返回博客列表" →   回到 blog.html
```

## 📊 数据结构设计

### posts.json 结构

```json
{
  "posts": [           // 文章数组
    {
      "id": "string",           // 唯一标识 (必需)
      "title": "string",        // 标题 (必需)
      "author": "string",       // 作者 (可选)
      "date": "YYYY-MM-DD",    // 日期 (必需)
      "tags": ["string"],       // 标签数组 (可选)
      "excerpt": "string",      // 摘要 (可选)
      "readTime": "string",     // 阅读时间 (可选)
      "content": "markdown"     // 内容 (必需)
    }
  ]
}
```

// blog.js 全局状态

## 🎯 核心算法

### 分页算法

```javascript
// 输入
const allPosts = [文章数组];
const currentPage = 当前页码;
const POSTS_PER_PAGE = 每页数量;

// 计算
const totalPages = Math.ceil(allPosts.length / POSTS_PER_PAGE);
const startIndex = (currentPage - 1) * POSTS_PER_PAGE;
const endIndex = startIndex + POSTS_PER_PAGE;

// 输出
const currentPosts = allPosts.slice(startIndex, endIndex);
```

### 页码生成算法

```javascript
// 智能页码显示
// 示例：[1] [2] [...] [5] [6] [7] [...] [10]

function getPageNumbers(current, total) {
  const maxVisible = 7;
  
  if (total <= maxVisible) {
    return [1, 2, 3, ..., total];
  }
  
  return [
    1,                           // 首页
    ...generateMiddlePages(),    // 中间页
    total                        // 末页
  ];
}
```

## 🔄 状态转换流程

```
用户操作 → 事件触发 → 状态更新 → 重新渲染

示例1: 翻页
点击"下一页" 
  → onclick="goToPage(currentPage+1)"
  → currentPage++
  → renderBlogList() + renderPagination()
  → 页面更新

示例2: 查看文章
点击文章标题
  → location.href = "post.html?id=xxx"
  → 页面跳转
  → loadPost()
  → 获取文章数据
  → renderPost()
  → 页面渲染
```
